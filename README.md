# edubot-2-Educational Assistant is an AI-powered web platform designed to help Kazakhstani students prepare for UNT exams. It provides 24/7 academic support through an intelligent chatbot, interactive courses, and realistic exam simulations.
 AI Chat Assistant - Instant answers to academic questions

 Interactive Courses - Math, Physics, Biology, History with progress tracking

 UNT Exam Simulator - Realistic test environment with 2500+ questions

 Responsive Design - Works perfectly on all devices

 Secure & Local - No API keys, fully client-side

 Modern UI - Clean, intuitive interface with animations


1. Ask Academic Questions
Type any question in the chat
Use quick buttons for common questions
Get detailed explanations with examples

2. Explore Courses
Click on subject cards
Track learning progress
Access lesson materials

4. Take Practice Tests
Click "Start Test" button
Answer 5 demo questions
View results and explanations
